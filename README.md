# Formulario de Suscripción - Frontend Basic

Este es un proyecto de práctica desarrollado como parte de mi formación en **Desarrollo Front-end**. Es un formulario de suscripción moderno, estético y funcional.

 Características
- Diseño Glassmorphism: Una tarjeta de formulario traslúcida que permite ver el fondo.
- Imagen de Fondo:.
- Validaciones Nativas: Uso de atributos required de HTML5 para garantizar que los datos sean completados.
- Estilos Personalizados: Botones con efectos a travez del uso del hover.
- Radio buttons personalizados con accent-color (SteelBlue).
- Tipografía clara y bordes de campos resaltados para mejor UX.

 Tecnologías utilizadas
- HTML5: Estructura semántica y validaciones.
- CSS3: Flexbox para centrado total, transparencias (RGBA) y diseño responsivo básico.
- Git & GitHub: Control de versiones y despliegue en GitHub Pages.
