# Formulario de Suscripción - Frontend Basics

Este proyecto es un formulario de suscripción moderno y estético, diseñado para practicar la estructura de formularios complejos y la interacción con el usuario mediante CSS.

##  Detalles Técnicos y Aprendizajes

- **Preparado para Backend:** El formulario está configurado con el atributo `method="POST"`, dejando la estructura lista para el envío de datos a un servidor o servicio de procesamiento, permitiendo la redirección automática a una página de agradecimiento o confirmación.
- **Identificación Precisa:** Uso de atributos `id` únicos en cada elemento del formulario para una vinculación correcta con sus etiquetas (`label`), mejorando la accesibilidad y la manipulación de datos.
- **Variedad de Inputs:** Implementación de distintos tipos de entrada, incluyendo botones de **Radio**, para ofrecer una experiencia de selección múltiple clara y variada.
- **Interactividad (Hover Effects):** Aplicación de estados `:hover` en botones y campos, proporcionando un feedback visual inmediato al usuario y mejorando la experiencia de navegación (UX).
- **Diseño Glassmorphism:** Una interfaz traslúcida sobre un fondo de imagen optimizado con `background-size: cover` para asegurar que la playa se vea bien en cualquier pantalla.

##  Tecnologías utilizadas
- **HTML5** (Semántica y tipos de input)
- **CSS3** (Flexbox, efectos de transición y selectores de ID)
- **Git & GitHub** (Flujo de trabajo profesional)
